Does this work for you? def mount_share(share_path): # Mounts a share at /Volumes, returns the mount point or raises an error sh_url = CoreFoundation.CFURLCreateWithString(None, share_path, None) # Set UI to reduced interaction # open_options = {NetFS.kNAUIOptionKey: NetFS.kNAUIOptionNoUI} # Allow mounting sub-directories of root shares # mount_options = {NetFS.kNetFSAllowSubMountsKey: True} # Mount! result, output = NetFS.NetFSMountURLSync(sh_url, None, None, None, None, None, None) # Check if it worked if result != 0: raise Exception(‘Error mounting url “%s”: %s’ % (share_path, output)) # Return the mountpath return str(output[0])

Got it: You also have to replace the mount/open options within the line: result, output = NetFS.NetFSMountURLSync(sh_url, None, None, None, open_options, mount_options, None) to result, output = NetFS.NetFSMountURLSync(sh_url, None, None, None, None, None, None)

This is exactly how it looks like and this worked for me. TY!

Hi Eric, i found it out. mount_options = { ‘AllowSubMounts’: True, ‘MountFlags’: 0x00100000 } # Mount! result, output = NetFS.NetFSMountURLSync(sh_url, None, None, None, None, mount_options, None) This will do a hidden mount. The difference in syntax between sierra and high sierra in interpreting the parameters: NetFS.kNAUIOptionKey: NetFS.kNAUIOptionNoUI (Sierra) ‘UIOption’: NoUI (High Sierra) Greetings from Germany!
